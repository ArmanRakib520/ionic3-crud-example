# IONIC 3 CRUD Example

A simple example of Ionic 3 CRUD  

## Technologies
* Ionic 3
* ExpressJS 4
* Mongoose 5
* NodeJS 8

## Installation

### API
```
cd api/
npm install
```
* Access api at http://localhost:8080

### CLIENT
```
npm install
ionic serve
```
* Access client at http://localhost:8100
